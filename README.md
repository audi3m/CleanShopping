## 프로젝트 정보
- 개발 기간
  - 2025.01.13 ~ 2025.01.25
- 개발 인원: 1인
- 지원 버전: iOS 17.0+
- 기술 스택 & 라이브러리
  - UI: UIKit
  - Data: SwiftData
- 앱 주요 기능
  - 네이버, 카카오 api를 활용한 도서 검색
  - 관심 있는 도서 저장
